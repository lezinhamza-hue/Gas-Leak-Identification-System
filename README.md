# Gas Leak Identification and Alert System

## Aim
To develop a gas leak detection system using Arduino and MQ2 gas sensor that alerts users using LED and buzzer when gas concentration exceeds safe limits.

## Components
- Arduino UNO
- MQ2 Gas Sensor
- LED
- Buzzer
- Resistor
- Breadboard

## Software
- Arduino IDE
- Tinkercad

## Working Principle
The MQ2 gas sensor detects the presence of combustible gases. When the gas level exceeds a predefined threshold, the Arduino activates a buzzer and LED to alert users.

Circuit simulation was developed using Tinkercad.
